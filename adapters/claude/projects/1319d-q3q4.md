## Repository Layout

```
1319D_Q3Q4/
├── realtek_a14/          # Main Android 14 SDK for RTD1319D (chip: parker)
│   ├── build.cfg         # Active build configuration (edit before building)
│   ├── build_def.cfg     # Default/fallback build config
│   ├── vendor/prime/     # PESI DTV middleware + HAL (prime overlay)

│   ├── device/realtek/   # Board configs (TRNE8_1319D_Hybrid is the target product)
│   ├── image_file/       # Image packing scripts and prebuilts
│   ├── flash_writer/     # Flash writer (rtd1319d_rtk_drm_smpv3.03.r01_nl)
│   ├── lk/               # Little Kernel (bootloader)
│   └── knowledge/        # Subsystem deep-dive notes (grow this as you learn)
```
